# TD4 エミュレータ

4ビットCPU「TD4」のエミュレータとアセンブラをブラウザ上で動かせるWebアプリケーションです。

## 機能

- **アセンブラ**: TD4のアセンブリ言語をマシンコードに変換
- **エミュレータ**: TD4のマシンコードをステップ実行・連続実行
- **入出力**: 4ビットの入力スイッチと出力LED表示
- **レジスタ表示**: A、B、PC、キャリーフラグの値を表示
- **メモリ表示**: ROM（16バイト）の内容を表示

## TD4 命令セット

| 命令 | 説明 |
|------|------|
| `ADD A, Im` | レジスタAにイミディエイト値を加算 |
| `ADD B, Im` | レジスタBにイミディエイト値を加算 |
| `MOV A, Im` | イミディエイト値をレジスタAに設定 |
| `MOV B, Im` | イミディエイト値をレジスタBに設定 |
| `MOV A, B` | レジスタBの値をAにコピー |
| `MOV B, A` | レジスタAの値をBにコピー |
| `IN A` | 入力ポートの値をレジスタAに読み込み |
| `IN B` | 入力ポートの値をレジスタBに読み込み |
| `OUT Im` | イミディエイト値を出力ポートに書き込み |
| `OUT B` | レジスタBの値を出力ポートに書き込み |
| `JMP Im` | 無条件ジャンプ |
| `JNC Im` | キャリーフラグが0の場合ジャンプ |
